# 🍷 יומן היין שלי — Wine Journal

יומן יין אישי חכם עם ניתוח AI. צלם תווית או הקלד שם יין — וקבל ניתוח סומלייה מלא.

## ✨ פיצ'רים

- 📸 **סריקת תוויות** — צלם תווית יין והבינה המלאכותית מזהה ומנתחת
- ⌨️ **חיפוש טקסט** — הקלד שם יין, יקב, או זן ענבים
- 🍷 **ניתוח סומלייה** — אזור, זנים, תיאור טעם, זיווג אוכל ועובדה מעניינת
- ⭐ **דירוג אישי** — דרג 1-5 כוכבים
- 📝 **רשמים** — תעד עם מי שתית, מה אכלת, מה חשבת
- 📊 **סטטיסטיקות** — כמה יינות, ממוצע דירוג, סוג מועדף
- 🏷️ **סינון** — לפי סוג יין (אדום, לבן, רוזה...)
- 💾 **שמירה מקומית** — הכל נשמר ב-localStorage במכשיר שלך

## 🚀 התקנה

### GitHub Pages
1. צור ריפו חדש ב-GitHub
2. העלה את כל הקבצים
3. Settings → Pages → Source: main branch
4. הכנס לכתובת שנוצרה

### שימוש מקומי
פשוט פתח את `index.html` בדפדפן.

## 🔑 הגדרות

בפתיחה ראשונה צריך להכניס מפתח Gemini API חינמי:
1. היכנס ל-[Google AI Studio](https://aistudio.google.com/apikey)
2. צור מפתח API
3. הדבק באפליקציה

המפתח נשמר מקומית בלבד ולא נשלח לשום מקום חוץ מ-Google.

## 🛠️ טכנולוגיות

- Vanilla HTML/CSS/JS — קובץ אחד, אפס תלויות
- Gemini 2.0 Flash API — ניתוח תמונות וטקסט
- localStorage — שמירה מקומית
- עברית RTL — ממשק מלא בעברית
