# 🍷 יומן היין שלי — Wine Journal

יומן יין אישי חכם עם ניתוח AI. צלם תווית או הקלד שם יין — וקבל ניתוח סומלייה מלא.

## ✨ פיצ'רים

- 📸 **סריקת תוויות** — צלם תווית יין והבינה המלאכותית מזהה ומנתחת
- ⌨️ **חיפוש טקסט** — הקלד שם יין, יקב, או זן ענבים
- 🍷 **ניתוח סומלייה** — אזור, זנים, תיאור טעם, זיווג אוכל ועובדה מעניינת
- ⭐ **דירוג אישי** — דרג 1-5 כוכבים
- 📝 **רשמים** — תעד עם מי שתית, מה אכלת, מה חשבת
- 📊 **סטטיסטיקות** — כמה יינות, ממוצע דירוג, סוג מועדף
- 🏷️ **סינון** — לפי סוג יין (אדום, לבן, רוזה...)
- 💾 **שמירה מקומית** — הכל נשמר ב-localStorage במכשיר שלך
